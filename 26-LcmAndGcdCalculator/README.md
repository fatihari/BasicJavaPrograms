## 26-LcmAndGcdCalculator

A program that finds the GCF (Greatest Common Factor) and LCM (Least Common Multiple) of the two entered numbers is designed. 

> GCF=(EBOB)
> 
> LCM=(EKOK)=(n1*n2)/GCF
